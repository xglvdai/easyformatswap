How-To Guide

📐 Unit Converter Guide

Four categories: Length · Weight · Temperature · Currency. Real-time bidirectional conversion — type and see results instantly.

Step-by-Step Instructions

1
Choose a category — Click the tabs at the top to switch between Length, Weight, Temperature, and Currency.
2
Select input and output units — Use the two dropdown menus to set the “From” (input) and “To” (output) units. Click the ↔ button to quickly swap them.
3
Enter a value — Type a number into the left input box and the converted result appears instantly on the right. You can also enter a value on the right side for reverse calculation.

Supported Units

CategoryUnits Supported
LengthMeter, Kilometer, Centimeter, Millimeter, Mile, Yard, Foot, Inch
WeightKilogram, Gram, Milligram, Pound, Ounce, Ton (metric)
TemperatureCelsius (°C), Fahrenheit (°F), Kelvin (K)
CurrencyUSD, EUR, GBP, JPY, CNY, KRW, AUD, CAD and other major currencies (real-time exchange rates)

Pro Tips

  • Bidirectional conversion: Both input fields are editable. The system detects which side you are typing on and outputs the result on the other side.
  • Quick swap: Click the ↔ button between the two dropdowns to instantly swap input and output units — no need to re-select manually.
  • Live currency rates: Currency conversions use public exchange rate APIs for real-time data, ensuring accurate conversions.
⚠️ Currency disclaimer: Exchange rates are sourced from a public API and are for reference only. Actual transactions should use your bank’s real-time rates. Loading currency data may require a network request.
