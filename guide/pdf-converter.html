📄 PDF Converter Guide

Three modes: PDF to Image · PDF to Text · Images to PDF. All processing runs locally in your browser.

Mode 1: PDF → Images

1
Select the “PDF → Images” tab — This mode is selected by default.
2
Drag in a PDF file — Drop your PDF onto the upload area or click to browse and select one.
3
Set the render scale — Higher values produce sharper images but larger files. 2× works well for most cases; 4× is suitable for printing.
4
Click Convert — Each page of the PDF becomes a separate PNG image. Download individually or as a batch.

Mode 2: PDF → Text

1
Select the “PDF → Text” tab — Click the tab to switch to text extraction mode.
2
Drag in a PDF file — Works with both native digital PDFs and scanned PDFs (digital PDFs produce better text results).
3
Click Convert — The tool extracts all text content and displays it as copyable plain text.
⚠️ Scanned PDFs: If your PDF is a scanned image without an embedded text layer, text extraction quality may be limited. Use high-quality scans or OCR preprocessing for better results.

Mode 3: Images → PDF

1
Select the “Images → PDF” tab — Click to switch to the image merge mode.
2
Drag in multiple images — Supports JPG, PNG, and WebP formats. Images appear in the order they are added.
3
Reorder (optional) — Drag and drop images in the file list to arrange their order in the final PDF.
4
Click Convert — All images are merged into a single PDF file and automatically downloaded.
✅ Tip: When merging ID photos, scanned documents, or screenshots, each image becomes its own full-sized page in the PDF, preserving original aspect ratios.
